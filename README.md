# baoyu-translate

Translates articles and documents between languages with three modes - quick (direct), normal (analyze then translate), and refined (analyze, translate, review, polish). Supports custom glossaries and terminology consistency via EXTEND.md. Use when user asks to "translate", "翻译", "精翻", "translate article", "translate to Chinese/English", "改成中文", "改

## Install

```bash
npx skills add https://github.com/Aston1690/baoyu-translate --yes
```

## Usage

After installing, this skill will be available in Claude Code and other compatible AI coding agents.
